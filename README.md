# ESPTimer
A timer based on ESP and 8x8 matrixes

# Hardware
## Electronics
ESP8266

Max7219 4x8x8 led matrix

Piezoelectric speaker

Two push buttons

## Case
3D printed custom design reused from other project.

# Software
Compile and flash with Arduino IDE.
